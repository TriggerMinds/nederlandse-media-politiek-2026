# 2026 Dossiers — Nederlandse Media & Politiek

**OSINT-verifieerbare dossiers over alle Nederlandse partijen, omroepen en kranten in 2026.**

---

## Repository structuur

```
├── politieke partijen 2026/     ← 17 fractie-dossiers (TK-zetelverdeling 2026)
├── nederlandse omroepen 2026/   ← 37 omroep-dossiers (publiek + commercieel)
│   ├── publiek/                 ← NPO, NOS, NTR, AVROTROS, BNNVARA, EO, etc.
│   └── commercieel/             ← RTL, Talpa, ESPN, Ziggo Sport
├── nederlandse kranten 2026/    ← 44 kranten-dossiers
│   ├── dpg-media/               ← AD, Volkskrant, Trouw, 8 regionale
│   ├── mediahuis/               ← Telegraaf, NRC, Metro, 8 regionale
│   └── overig/                  ← FD, RD, ND, NU.nl, ANP, GeenStijl, Dumpert
├── README.md
└── .gitignore
```

## Werkwijze

- Alle data is verzameld uit uitsluitend **openbare, controleerbare bronnen**
- Prioriteit: officiële websites (tweedekamer.nl, nrc.nl, volkskrant.nl, ad.nl, etc.)
- Fallback: Wikipedia, Villamedia, NOS Journaal
- Data gemarkeerd als `actueel`, `historisch` of `niet bevestigd`
- **Geen verzonnen data — geen aannames**
